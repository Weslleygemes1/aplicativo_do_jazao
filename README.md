# aplicativo_do_jazao
O que é o projeto?
Um app de sorteamento de cara ou coroa, com contagem de vitórias.

status: concluido

funcionalidades: sorteio de cara ou coroa, escolha de cara ou coroa, contagem de vitórias, e moeda animada.

tecnologias utilizadas:
kotlin, Android Studio panda 3 (2025.3.3).
projeto: empty views activity, api 29 (android 10), emulador Medium Phone api 36.1
bibliotecas: Glide 4.16.0
